Time study

 Approach to calculating the time to complete a job that relies on breaking a job
  down into work elements and timing how long each one takes

 Standard time: time required for a qualified worker to perform a job using a
  standard method under normal conditions

      Qualified worker: fully trained for the job

 Standard method: the method on which the standard time is based ­ ideally, the
  "best" method.

      Methods are described as a sequence of elements

Time study requirements

 What types of jobs are best suited for a time study?

      Job involves manual labor
      Job involves repetitive cycles
      Focus on one person

 Operator should be thoroughly familiar with the job (qualified worker)
 Standard method must be used
 Operator and supervisor must know that the job is being studied

      Need to obtain approval from both
      Analyst should not be biased and should refrain from any operator criticism

Identify breakpoints for your elements

 Make sure your work elements are not less than

      0.04 minutes (~3 seconds)

 Elements might be regular or irregular

      Regular element: happens every cycle
      Irregular element: does not happen every cycle

 To make elements easier to isolate, identify both sight and sound break points.

Dealing with abnormal elements

 Indicate abnormal elements that occur with a circled label (A, B, etc.), place it where it occurred,
  and explain in the footnotes

 Case 1: If it occurred in between two normal elements, record in the footnotes for completeness
  and ignore

 Case 2: If it occurred as part of a normal element

       Case 2a: If the abnormal element is long enough, record its time separately below the table and later
         subtract it from the element time

       Case 2b: If it is not, just record it as part of the normal element time and circle that element time.
         Then, we determine if the time is an outlier

Sensory memory

 Stores stimuli detected by sensory receptors:

       Sensory memory forms automatically, without attention or interpretation
       However, attention is needed to transfer information to working memory to support perceptual process

 Large capacity - Can hold many items at once, but information is "unprocessed" (no
  encoding) based on recognition through use of LTM.

 Short-duration buffer with rapid decay

 Three types (with varying duration):

       Iconic memory:

                Visual stimuli
                Usually lasts about 0.3 seconds (Efron, 1970)

       Echoic memory

                Sensory memory for auditory input
                Lasts only 2 to 3 seconds

       Kinesthetic

                Tactile stimuli
                Lasts for several seconds after contact

Unitization

 When stimulus is perceived as "whole" (vs. sum of parts)

      As familiarity with perceptual pattern increases, stimulus processing transform from feature
        analysis to global or holistic processing (Gestalt of image)

      Allows humans to read familiar words and phrases rapidly and overlook typographical errors
      Gestalt is a theory of visual perception with principles identifying specific forms of perceptual

        processing

             Similarity occurs when there are similar features of co-located but different stimuli
             Continuation occurs when the eye moves through one object to complete another
             Closure occurs when objects are incomplete or space is not enclosed
             Anomaly detection occurs when one part is dissimilar from others comprising the "whole"

Top-down processing

 Prior experiences and expectations determine what you see

      e.g., strong familiarity with image of Lincoln leads to identification of face; absence of
        familiarity lead to identification of figure of woman at center of image

Top-down and bottom-up processing implications for interface
design

 Maximize bottom-up processing

      Make features easy to identify using size and contrast
      Highlight patterns or feature cueing pattern (e.g.,

        compass pointer)

 Maximize top-down processing

      Use words, as goals are typically verbally encoded in WM
      Minimize number of words that need to be recognized
      Provide contextual information

              e.g., grouping of icons according to function

 Maximize automaticity and unitization:

      Use familiar perceptual representations (those
        encountered frequently and stored in LTM)

      e.g., meaningful icons (clock face, calculator functions),
        familiar fonts and mixed text, and words rather than
        abbreviations

Working memory (WM)

 Role: Keeps information active while in use

 Not a synonym for short term memory!

      Refers to short term store needed for certain mental tasks
      Defined not in terms of duration, but rather in terms of purpose

 Short-term memory (STM) usually refers to the temporary storage of a small amount of
  information for a brief period (seconds to minutes) without active manipulation.

      Example: remembering a phone number just long enough to dial it.
      It's more about holding information.

 Working memory (WM) includes short-term storage plus the active processing and
  manipulation of that information to achieve a goal

      Example: remembering a number while also calculating its sum with another number. It's more
        about holding and working with information.

Working memory (WM)

 Miller's chunking theory: 7▒2 "chunks" of information is the limit of the working memory.

 A chunk is a meaningful unit of information created by grouping smaller bits together using prior knowledge.

        Example: Remembering 1 9 4 5 as "1945" (a year) instead of four separate digits.
        Example: In language, the word "dog" is a single chunk rather than three separate letters.

 Chunking works because WM leverages LTM to combine related elements into a single, meaningful unit,
   effectively expanding what you can store.

 Experts in a domain can remember more complex patterns because they have richer, more meaningful chunks
   in long-term memory (e.g., a chess master seeing whole board configurations rather than isolated pieces).

 WM is often described as a bottleneck for cognition

        Only a few chunks can be actively maintained and manipulated at once.

 If information in WM (or STM) is not actively rehearsed, it fades within about 15ş30 seconds.

 If you have more chunks to hold, each chunk gets less rehearsal time, making decay more likely.

        The total time before an item is lost depends on how quickly you cycle through the items during rehearsal.

Long-term memory (LTM)

 Role: Keeps event information available for recall over long periods of time

 Key characteristics:

      A storage of information in the brain for a long or an indefinite period.
      Simplifies working memory processing
      Establishes expectations about the world
      Capacity and duration are unlimited.

 Process:

      Item strength = f { frequency, recency, importance }
      Forgetting (weak strength, few associations, interfering associations)

 Short term memories can become long term memories through rehearsal and/or
  encoding information semantically (i.e. based on meaningful association

 Long-term memory

Guidelines:
 Don't violate people's mental framework, i.e., design according to human expectations.
 Place most important information at the beginning or end.
 Relate new information to previous information (to integrate into existing frameworks).
 Appeal to the emotional component of memory by using humor and emotion.
 Storytelling: helps provide a framework and also appeals to emotion.

 Memory aids (supplement LTM)
 Carefully designed information (limit LTM dependence, support associations)
 Reminders can be used to avoid failures of prospective memory

 Attention is the process of selecting which information to process.

             Channel switching
             Shining a flashlight in a dark attic

 Attention is required to support most cognitive activities (not LTM or sensory registers)

      Attention as a limited resource.

Types of Attention

 Selective/Focused Attention

      Maintain attention on one or a few sources of information and not be distracted by other
        sources

 Divided Attention

      Required to do more than one task at the same time (e.g., cocktail party effect)

 Sustained Attention

      The ability of observers to maintain attention and remain alert to stimuli over prolonged
        periods of time (i.e., vigilance)

Selective Attention: Design Implications

 Role: Ability to focus on a specific set of stimuli.

      Selective attention does not guarantee perception but is a necessary condition.

 High value information sources should require minimal effort to attend

 Important, unexpected or rare events should be conveyed in a salient manner

n(r) in 2 Minutes

Fast exam version: the formula always starts from the same definition. Only the pdf and support change.

\[ n(r)=E[(X_L-r)^+]=\int_r^{\infty}(x-r)f_L(x)\,dx \]
If demand is normal, use the shortcut n(r) = sigma_L L(z). If demand is not normal, go back to the integral.

Exam Shortcuts

  1. Write the support of demand during lead time.
  2. Start from n(r) = integral of (x-r)f(x).
  3. Trim the limits to the actual support.
  4. If r is below the whole support, use n(r)=E[X_L]-r.
  5. If the problem gives Type II service level, use n(r)=(1-beta)q.

For a discrete distribution, replace the integral with sum over x>r of (x-r)P(X=x).

General Recipe

  1. Identify the lead-time demand random variable X_L.
  2. Write its pdf f_L(x) and support.
  3. Start with:
    \[ n(r)=\int_r^{\infty}(x-r)f_L(x)\,dx \]
  4. Replace the limits with the actual support.
  5. Split into cases if r can be below, inside, or above the support.
  6. Integrate and simplify.

Why the Limits Change

\[ n(r)=\int_r^{\infty}(x-r)f_L(x)\,dx \]

If the support is [a,b], then f_L(x)=0 outside that interval, so:

\[ n(r)=\int_{\max(r,a)}^b (x-r)f_L(x)\,dx \]
This is the key idea for uniform distributions and any other bounded distribution.
